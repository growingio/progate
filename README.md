# Progate

## 模块
| 名称        | 功能           |
|:----------|--------------|
| api       | 接口定义         |
| utilities | 工具类          |
| compiler  | 编译器          |
| graphql   | GraphQL 入口实现 |
| grpc      | gRPC 出口实现    |
| core      | 核心（负责组织各个模块） |
| bootstrap | 程序启动器        |

## Upstream

### gRPC
